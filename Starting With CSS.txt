Assignment- Starting with CSS

1. What is CSS and why use it?
Ans:  CSS - Cascading Style Sheets is used to create good-looking websites by describing how the HTML elements are displayed on the screen. It also controls the layout of the website. 


2. What are the different ways to bring CSS into an HTML file?
Ans: We can add CSS to HTML in 3 ways. Those are as follows:-
1. Inline styling.
2. Internal styling.
3. External styling.


3. What do you mean by specificity in CSS?
Ans: Specificity in CSS is a mechanism that determines which style rule applies to an element when multiple rules could potentially match. It's like a competition between different CSS rules, and the one with the highest "weight" or specificity wins.
